# Requests a new random number generation

 

**URL** : `/random`

**Example**: `http://127.0.0.1:8080/number-generator/api/random`

**Method** : `POST`

## Success Response

**Code** : `200 OK`

 

**Example**

````json

{
    "code": 200,
    "description": "The random number request was submitted sucessfully",
    "message": "SUCCESS"
}

````
